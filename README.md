## What's Included

### 1. piComp - Batch Image Compression Tool

piComp is a powerful CLI tool for batch image compression. Reduce multiple images in one go with automatic resizing and quality optimization.

**Features:**
- Batch compress all images in a folder with one command
- Automatically resize images to max dimensions
- Supports JPG, PNG, WebP, GIF
- Shows compression statistics and space saved
- Built on sharp for maximum speed

### 2. fileRen - Batch File Renaming Tool

fileRen is a powerful CLI tool for batch file renaming. Rename multiple files with patterns, prefixes, suffixes and more.

**Features:**
- Add prefix/suffix to all files
- Replace text in filenames
- Auto-numbered renaming
- Convert to lowercase/uppercase
- Filter by file extensions
